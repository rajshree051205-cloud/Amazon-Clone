# 🛒 Amazon Clone (HTML & CSS)

A responsive front-end clone of the Amazon homepage built using **HTML** and **CSS**.
This project focuses on layout design, styling, and replicating a real-world e-commerce UI.

## 📌 Features

* ✅ Amazon-like homepage UI
* ✅ Hero section with banner
* ✅ Product category boxes
* ✅ Hover effects for better UI experience
* ✅ Clean and structured layout
* ✅ Fully built using HTML & CSS (no JavaScript)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Flexbox
* Basic responsive design
